# 黑暗之魂系列文本数据库 - 重构方案

## 一、重构目标

将现有的纯静态HTML/XML项目重构为现代化的Web应用，提升性能、可维护性和用户体验。

### 核心目标
- ✅ 采用现代前端框架和工具链
- ✅ 优化数据加载和渲染性能
- ✅ 提升代码可维护性和可扩展性
- ✅ 增强用户交互体验
- ✅ 支持PWA离线访问
- ✅ 优化移动端体验

## 二、技术栈选型

### 2.1 推荐方案：Vue 3 + Vite

**选择理由**：
- **Vue 3**: 渐进式框架，学习曲线平缓，适合中小型项目
- **Vite**: 极速的开发服务器，优秀的构建性能
- **TypeScript**: 类型安全，提升代码质量
- **轻量级**: 打包体积小，适合静态部署

### 2.2 完整技术栈

```
前端框架: Vue 3 (Composition API)
构建工具: Vite 5
语言: TypeScript
状态管理: Pinia
路由: Vue Router 4
UI组件: 
  - Element Plus (可选，用于复杂组件)
  - 或自定义组件（保持原有风格）
样式方案: 
  - SCSS/LESS (CSS预处理器)
  - CSS Modules (样式隔离)
数据格式: JSON (替代XML)
图标: WebP (保持现有格式)
PWA: Vite PWA Plugin
部署: GitHub Pages / Vercel / Netlify
```

### 2.3 备选方案

#### 方案A: React + Next.js
- 适合需要SSR的场景
- 生态更丰富
- 学习成本较高

#### 方案B: Svelte + SvelteKit
- 编译时框架，运行时体积最小
- 性能最优
- 生态相对较小

## 四、数据迁移方案

### 4.1 XML转JSON

创建转换脚本，将XML数据转换为JSON格式：

```javascript
// scripts/convertXmlToJson.js
const fs = require('fs');
const xml2js = require('xml2js');
const path = require('path');

async function convertXmlToJson(xmlPath, jsonPath) {
  const xmlData = fs.readFileSync(xmlPath, 'utf-8');
  const parser = new xml2js.Parser();
  
  const result = await parser.parseStringPromise(xmlData);
  const items = result.entries.text.map(item => ({
    id: item.$.id,
    text: item._
  }));
  
  fs.writeFileSync(jsonPath, JSON.stringify(items, null, 2));
}
```

## 五、核心功能实现

### 5.1 数据加载策略

```typescript
// composables/useGameData.ts
import { ref, computed } from 'vue';
import type { Item } from '@/types/item';

export function useGameData(game: 1 | 2 | 3, type: string) {
  const items = ref<Item[]>([]);
  const loading = ref(false);
  const error = ref<Error | null>(null);

  // 懒加载数据
  const loadData = async () => {
    loading.value = true;
    try {
      const module = await import(`@/data/ds${game}/${type}s.json`);
      items.value = module.default;
    } catch (e) {
      error.value = e as Error;
    } finally {
      loading.value = false;
    }
  };

  return {
    items,
    loading,
    error,
    loadData
  };
}
```

### 5.2 搜索功能

```typescript
// composables/useSearch.ts
import { ref, computed } from 'vue';
import type { Item } from '@/types/item';

export function useSearch(items: Ref<Item[]>) {
  const searchQuery = ref('');
  const searchLanguage = ref<'chn' | 'jap' | 'eng'>('chn');

  const filteredItems = computed(() => {
    if (!searchQuery.value) return items.value;
    
    const query = searchQuery.value.toLowerCase();
    return items.value.filter(item => {
      const name = item.name[searchLanguage.value].toLowerCase();
      const desc = item.description[searchLanguage.value].toLowerCase();
      return name.includes(query) || desc.includes(query);
    });
  });

  return {
    searchQuery,
    searchLanguage,
    filteredItems
  };
}
```

### 5.3 收藏功能

```typescript
// composables/useFavorite.ts
import { ref, watch } from 'vue';

export function useFavorite() {
  const favorites = ref<Set<string>>(new Set());

  // 从localStorage加载
  const loadFavorites = () => {
    const saved = localStorage.getItem('ds-favorites');
    if (saved) {
      favorites.value = new Set(JSON.parse(saved));
    }
  };

  // 保存到localStorage
  watch(favorites, (newFavorites) => {
    localStorage.setItem('ds-favorites', 
      JSON.stringify([...newFavorites]));
  }, { deep: true });

  const toggleFavorite = (id: string) => {
    if (favorites.value.has(id)) {
      favorites.value.delete(id);
    } else {
      favorites.value.add(id);
    }
  };

  const isFavorite = (id: string) => favorites.value.has(id);

  return {
    favorites,
    loadFavorites,
    toggleFavorite,
    isFavorite
  };
}
```

### 5.4 文本格式化

```typescript
// utils/formatter.ts
export function formatGameText(text: string): string {
  return text
    .replace(/#1/g, '<span class="text-gold line-through">')
    .replace(/#2/g, '<span class="text-gold">')
    .replace(/#3/g, '</span>')
    .replace(/##/g, '<i class="text-red">未使用</i>')
    .replace(/#0/g, '<strong class="text-title">')
    .replace(/\n/g, '<br>');
}

export function stripFormatting(text: string): string {
  return text.replace(/(#1|#2|#3|##|#0)/g, '');
}
```
