### 3. 视觉风格 ✅

#### 配色方案
```scss
背景: #000 (纯黑) + #111 (深灰)
边框: #430 (深棕) + #531 (棕色)
文字: #ccc (浅灰) + #aaa (灰色)
强调: #960 (金棕色)
特殊: #fe6 (金色) + #f00 (红色)
```

#### 布局特点
- 固定宽度 1800px 居中
- 三语并排（中/日/英各 28%）
- 图标左侧 10% 宽度
- 内嵌阴影 `box-shadow: inset 0 0 .3em .1em #531`
- 固定顶部导航 `position: fixed`

#### 字体设置
- 中文：仿宋 (SimSun)
- 英文：Palatino Linotype
- 字号：18px
- 行高：1.5

---

### 4. 特殊文本样式 ✅

#### 支持的标记
- `[uud]...[/uud]` - 未使用文本（红色删除线）
- `[cgb]...[/cgb]` - 已改变文本（黄色删除线）
- `[cgd]...[/cgd]` - 金色文本
- `[dtt]...[/dtt]` - 标题文本（渐变背景）

#### 兼容旧格式
- `#1` → 黄色删除线
- `#2` → 金色文本
- `#3` → 结束标记
- `##` → 未使用标记
- `#0` → 标题文本
