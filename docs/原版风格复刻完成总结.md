### 3. 全局语言切换集成

#### 导航栏语言切换

在 `ImprovedNavigation.vue` 中实现：

**桌面端**：
- 导航栏右侧语言按钮（中/日/EN）
- 使用 border-left 分隔
- 激活状态高亮显示

**移动端**：
- 菜单底部语言选择区域
- 三个按钮垂直排列
- 显示完整语言名称

#### 状态管理

使用 `userStore.currentLanguage` 管理全局语言状态：

```typescript
const displayName = computed(() => {
  if (!item.value) return '';
  return item.value.name[userStore.currentLanguage];
});

const displayDescription = computed(() => {
  if (!item.value) return '';
  return item.value.description[userStore.currentLanguage];
});

const displayRemark = computed(() => {
  if (!item.value || !item.value.remark) return '';
  return item.value.remark[userStore.currentLanguage] || '';
});
```

#### 语言持久化

- 使用 localStorage 保存用户语言偏好
- 页面刷新后自动恢复上次选择的语言
- 语言切换实时生效，无需刷新页面

## 技术要点

### 1. 垂直居中技术

```css
.desfrm {
  transform: translateY(-50%);
  margin: 12em 0;
}
```

**原理**：
- 设置大的上边距（12em）
- 使用 transform 向上偏移 50%
- 实现视觉上的垂直居中
- 内容可以自适应高度

### 2. 深度选择器

```scss
.desfrm {
  :deep(p) {
    border-bottom: 1px solid #321;
    // ...
  }
}
```

**用途**：
- 样式化 v-html 渲染的内容
- 确保分隔线正确显示
- 保持样式封装性

### 3. 图片错误处理

```typescript
const handleImageError = (e: Event) => {
  const img = e.target as HTMLImageElement;
  img.style.display = 'none';
};
```

**作用**：
- 图片加载失败时隐藏图标
- 避免显示破损图片图标
- 提升用户体验
