## 技术实现

### 配色方案
```scss
$color-bg-primary: #000;      // 纯黑背景
$color-bg-secondary: #111;    // 深灰背景
$color-border: #430;          // 深棕边框
$color-border-light: #531;    // 棕色边框
$color-accent: #960;          // 金棕色强调
$color-text-primary: #ccc;    // 浅灰文字
$color-text-secondary: #aaa;  // 灰色文字
$color-text-gold: #fe6;       // 金色文字
$color-text-red: #f00;        // 红色文字
```
