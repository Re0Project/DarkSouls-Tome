# 原版风格复刻方案

## 目标

在当前 Vue 3 项目基础上，完全复刻原版的视觉风格和交互体验。

---

## 原版风格分析

### 1. 颜色方案
```css
背景色:
- 主背景: #000 (纯黑)
- 次背景: #111 (深灰)
- 卡片背景: #111

边框色:
- 主边框: #430 (深棕)
- 次边框: #531 (棕色)

文字色:
- 主文字: #ccc (浅灰)
- 次文字: #aaa (灰色)
- 强调色: #960 (金棕色)
- 金色: #fe6
- 红色: #f00

滚动条:
- 背景: #222
- 滑块: #960
```

### 2. 布局特点
- **固定宽度**: 1800px 居中布局
- **三语并排**: 中文/日文/英文 各占 28% 宽度
- **图标左侧**: 10% 宽度，图标居中显示
- **内嵌阴影**: `box-shadow: inset 0 0 .3em .1em #531`
- **固定导航**: position: fixed, 半透明背景

### 3. 字体
- **中文**: 仿宋
- **英文**: Palatino Linotype
- **日文**: 默认字体
- **字号**: 18px
- **行高**: 1.5

### 4. 特殊样式
- **删除线文本**: 
  - 未使用: 红色删除线 `rgba(255,0,0,.4)`
  - 已改变: 黄色删除线 `rgba(255,238,102,.8)`
- **标题样式**: 渐变背景 `linear-gradient(to right, rgba(80,48,1,0), #531, rgba(80,48,1,0))`
- **悬停效果**: 颜色变为 #960

### 5. 交互特点
- **三语切换**: 移动端点击切换显示语言
- **图标悬停**: 边框变为 5px solid #531
- **平滑过渡**: transition: .3s
- **返回顶部**: 右下角三角形按钮

### 阶段四：特殊文本样式

#### 4.1 更新 formatter.ts
```typescript
// src/utils/formatter.ts
export function formatText(text: string): string {
  if (!text) return '';
  
  let formatted = text;
  
  // 未使用文本 - 红色删除线
  formatted = formatted.replace(
    /\[uud\](.*?)\[\/uud\]/g,
    '<span class="uud">$1</span>'
  );
  
  // 已改变文本 - 黄色删除线
  formatted = formatted.replace(
    /\[cgb\](.*?)\[\/cgb\]/g,
    '<span class="cgb">$1</span>'
  );
  
  // 金色文本
  formatted = formatted.replace(
    /\[cgd\](.*?)\[\/cgd\]/g,
    '<span class="cgd">$1</span>'
  );
  
  // 标题文本
  formatted = formatted.replace(
    /\[dtt\](.*?)\[\/dtt\]/g,
    '<span class="dtt">$1</span>'
  );
  
  // 换行
  formatted = formatted.replace(/\n/g, '<br>');
  
  return formatted;
}
```

1. **Vue Router** - 单页应用路由

3. **TypeScript** - 类型安全
4. **懒加载** - 图片懒加载

这些功能在底层工作，不影响视觉风格。
