## CSS 样式详解

### 1. 容器样式

#### `.contentd` - 主容器
```css
.contentd {
  width: 1800px;
  margin: 0 auto;
  overflow: hidden;
}
```

**作用**：
- 固定宽度 1800px
- 居中显示
- 隐藏溢出内容

---

### 2. 框架样式

#### `.frame` - 物品框架
```css
.frame {
  text-align: center;
  overflow: hidden;
  background: #111;
  padding: 3em 2em;
  margin: 1em 0 2em;
  box-shadow: inset 0 0 .3em .1em #531;
  font: 18px/1.5 仿宋;
  color: #ccc;
}
```

**特点**：
- 深灰背景 `#111`
- 内嵌阴影效果 `inset 0 0 .3em .1em #531`
- 仿宋字体，18px 行高 1.5
- 居中对齐

---

### 3. 图标样式

#### `.icon` - 图标容器
```css
.icon {
  float: left;
  width: 10%;
  height: 1px;
}

.icon img {
  transform: translateY(-50%);
  margin: 14em 0 0;
  max-width: 120%;
}
```

**布局原理**：
- 左浮动，占 10% 宽度
- 高度设为 1px（最小高度）
- 图片使用 `translateY(-50%)` 垂直居中
- 上边距 14em 用于定位
- 最大宽度 120%（允许图片略微超出容器）

**为什么这样设计？**
- 图标需要在左侧固定位置
- 使用 `transform` 实现精确的垂直居中
- 大边距配合 transform 创造视觉平衡

---

### 4. 三语布局

#### 桌面端 - 三列并排

```css
.chn, .eng, .jap, .title {
  float: left;
  width: 28%;
  padding: 1%;
}

.contentd .chn, .contentd .eng, .contentd .jap {
  width: 31.3%;
}
```

**布局计算**：
- 每列约 31.3% 宽度
- 三列总计约 94%（留出边距）
- 左浮动实现并排

#### 标题样式

```css
.title {
  width: 88%;
}

.contentd .title {
  padding: 1% 1% 2%;
}

.name {
  height: 2em;
  line-height: 2em;
  font-weight: 700;
  padding: 0;
}
```

**特点**：
- 标题占 88% 宽度（横跨三列）
- 固定高度 2em
- 加粗显示

---

### 5. 描述区域

#### `.desc` - 描述容器
```css
.desc {
  height: 24em;
}

.desfrm {
  transform: translateY(-50%);
  margin: 12em 0;
}

.desfrm p {
  border-bottom: 1px solid #321;
  box-sizing: border-box;
  min-height: 1.5em;
}

.desfrm p:last-child {
  border-bottom: 0;
}
```

**布局原理**：
- 描述区域固定高度 24em
- 内容使用 `translateY(-50%)` 垂直居中
- 上边距 12em 配合 transform 实现居中
- 每段文字底部有分隔线
- 最后一段无分隔线

**为什么用 transform？**
- 实现精确的垂直居中
- 配合固定高度创造视觉平衡
- 内容可以自适应高度

---

### 6. 备注区域

#### `.remk` - 备注容器
```css
.remk {
  height: 3em;
}

.remk p {
  transform: translateY(-50%);
  margin: 1.5em 0;
}
```

**特点**：
- 固定高度 3em
- 内容垂直居中
- 较小的上下边距

---

### 7. 字体样式

```css
.eng {
  font-family: Palatino Linotype;
}

.contentd .name {
  font-family: Palatino Linotype, 仿宋;
}
```

**字体选择**：
- 中文/日文：仿宋
- 英文：Palatino Linotype（衬线字体）
- 名称：混合字体（支持多语言）

## 移动端适配

### 响应式断点

```css
@media screen and (max-width:1000px) {
  /* 移动端样式 */
}
```

### 移动端布局变化

#### 1. 容器适配
```css
.contentd {
  width: 100%;
  overflow: auto;
}

.frame {
  font: 8px/1.5 仿宋;
  margin: 0 0 10em;
}
```

#### 2. 图标居中
```css
.icon {
  float: none;
  width: 33%;
  height: auto;
  margin: auto;
}

.icon img {
  transform: translateY(0);
  margin: 0;
}
```

**变化**：
- 取消浮动
- 宽度改为 33%
- 图片不再使用 transform
- 边距清零

#### 3. 三语切换

```css
.chn, .eng, .jap, .title {
  width: 33% !important;
  padding: 5% 0;
  z-index: 1;
}

.name, .desc, .remk, .contentd .desfrm {
  width: 300%;
  opacity: 0;
  transition: .3s;
}
```

**切换原理**：
- 每个语言块宽度 33%（三个并排）
- 内容宽度 300%（三倍宽度）
- 默认透明度 0（隐藏）

#### 4. 语言切换按钮

```css
.tab {
  display: block;
  margin: 2em;
  line-height: 2;
  border: 1px solid #430;
  transition: .3s;
}
```

**移动端显示**：
- 桌面端隐藏（`display: none`）
- 移动端显示为块级元素
- 边框和过渡效果

#### 5. 激活状态

```css
.frame.c .chn,
.frame.j .jap,
.frame.e .eng {
  z-index: 2;
  position: relative;
}

.frame.c .chn div,
.frame.j .jap div,
.frame.e .eng div,
.contentd .name {
  opacity: 1 !important;
}

.frame.c .chn .tab,
.frame.j .jap .tab,
.frame.e .eng .tab {
  background: #531;
}
```

**状态类**：
- `.frame.c` - 中文激活
- `.frame.j` - 日文激活
- `.frame.e` - 英文激活

**效果**：
- 激活的语言 z-index 提升
- 内容透明度变为 1（显示）
- 按钮背景高亮

#### 6. 内容偏移

```css
.jap > div {
  margin-left: -100% !important;
}

.eng > div {
  margin-left: -200% !important;
}
```

**偏移原理**：
- 中文：margin-left: 0（默认）
- 日文：margin-left: -100%（左移一个宽度）
- 英文：margin-left: -200%（左移两个宽度）

**实现效果**：
- 三个语言块横向排列
- 通过负边距实现滑动切换
- 配合透明度实现淡入淡出

## 关键技术点

### 1. 垂直居中技术

```css
transform: translateY(-50%);
margin: 12em 0;
```

**原理**：
- 设置大的上边距
- 使用 transform 向上偏移 50%
- 实现视觉上的垂直居中

### 2. 三语切换技术

**桌面端**：
- 三列并排（float: left）
- 每列约 31.3% 宽度

**移动端**：
- 内容宽度 300%（三倍）
- 使用负边距偏移（-100%, -200%）
- 透明度控制显示/隐藏
- 状态类控制激活语言

### 3. 响应式设计

**断点**：1000px

**变化**：
- 容器：1800px → 100%
- 字体：18px → 8px
- 布局：三列并排 → 单列切换
- 图标：左浮动 → 居中
- 导航：固定宽度 → 100%

## 总结

原版物品详情页的设计特点：

1. **固定宽度布局**：1800px 居中显示
2. **左右分栏**：图标 10% + 内容 90%
3. **三语并排**：桌面端三列，移动端切换
4. **垂直居中**：使用 transform + margin 技巧
5. **内嵌阴影**：深色背景 + 棕色内阴影
6. **响应式设计**：1000px 断点，完全不同的布局
7. **精致细节**：分隔线、悬停效果、字体选择

**核心技术**：
- CSS Float 布局
- Transform 垂直居中
- 负边距实现滑动
- 透明度控制显示
- 状态类管理激活
