# 一览表设计方案

## 参考来源

艾尔登法环WIKI - 物品一览页面设计

## 设计目标

1. 保持原版黑魂风格（纯黑背景、金棕色）
2. 添加侧边栏分类导航
3. 实现一览表视图
4. 实现物品详情页
5. 响应式设计（桌面端 + 移动端）

## 组件设计

### 1. 侧边栏导航（SidebarNav）

#### 功能
- 显示物品分类
- 高亮当前分类
- 可折叠/展开
- 移动端可隐藏

#### 结构
```vue
<template>
  <aside class="sidebar-nav">
    <div class="nav-section">
      <h3 class="section-title">物品</h3>
      <ul class="nav-list">
        <li v-for="type in itemTypes" :key="type.value">
          <router-link 
            :to="`/ds${game}/${type.value}`"
            :class="{ active: currentType === type.value }"
          >
            {{ type.label }}
          </router-link>
        </li>
      </ul>
    </div>
  </aside>
</template>
```

### 2. 物品一览表（ItemCatalogTable）

#### 功能
- 表格形式展示物品
- 显示图标、名称、类型等信息
- 点击跳转到详情页
- 支持排序和筛选

#### 结构
```vue
<template>
  <div class="item-catalog-table">
    <div class="table-header">
      <h2>{{ title }}</h2>
      <div class="table-controls">
        <input 
          v-model="searchQuery" 
          type="text" 
          placeholder="搜索..."
          class="search-input"
        >
      </div>
    </div>
    
    <table class="catalog-table">
      <thead>
        <tr>
          <th>图标</th>
          <th>名称</th>
          <th>类型</th>
          <th>描述</th>
        </tr>
      </thead>
      <tbody>
        <tr 
          v-for="item in filteredItems" 
          :key="item.id"
          @click="goToDetail(item.id)"
          class="table-row"
        >
          <td class="icon-cell">
            <img :src="`/icons/${item.icon}`" :alt="item.name.chn">
          </td>
          <td class="name-cell">{{ item.name.chn }}</td>
          <td class="type-cell">{{ item.type }}</td>
          <td class="desc-cell">{{ truncate(item.description.chn) }}</td>
        </tr>
      </tbody>
    </table>
  </div>
</template>
```

### 3. 物品详情页（ItemDetailPage）

#### 功能
- 显示物品完整信息
- 大图标展示
- 三语对照描述
- 相关物品推荐
- 收藏功能

#### 结构
```vue
<template>
  <div class="item-detail-page">
    <div class="detail-container">
      <!-- 面包屑导航 -->
      <nav class="breadcrumb">
        <router-link to="/">首页</router-link>
        <span>/</span>
        <router-link :to="`/ds${game}/${type}`">{{ typeName }}</router-link>
        <span>/</span>
        <span>{{ item.name.chn }}</span>
      </nav>
      
      <!-- 物品信息 -->
      <div class="detail-content">
        <div class="detail-header">
          <div class="item-icon-large">
            <img :src="`/icons/${item.icon}`" :alt="item.name.chn">
          </div>
          <div class="item-meta">
            <h1>{{ item.name.chn }}</h1>
            <p class="item-type">{{ typeName }}</p>
            <button @click="toggleFavorite" class="favorite-btn">
              {{ isFavorite ? '★ 已收藏' : '☆ 收藏' }}
            </button>
          </div>
        </div>
        
        <!-- 三语描述 -->
        <div class="detail-descriptions">
          <div class="desc-section">
            <h3>中文</h3>
            <div v-html="formatText(item.description.chn)"></div>
          </div>
          <div class="desc-section">
            <h3>日本語</h3>
            <div v-html="formatText(item.description.jap)"></div>
          </div>
          <div class="desc-section">
            <h3>English</h3>
            <div v-html="formatText(item.description.eng)"></div>
          </div>
        </div>
        
        <!-- 备注 -->
        <div v-if="item.remark" class="detail-remark">
          <h3>备注</h3>
          <p v-html="formatText(item.remark.chn)"></p>
        </div>
        
        <!-- 相关物品 -->
        <div class="related-items">
          <h3>相关物品</h3>
          <div class="related-grid">
            <router-link 
              v-for="related in relatedItems" 
              :key="related.id"
              :to="`/ds${game}/${type}/${related.id}`"
              class="related-item"
            >
              <img :src="`/icons/${related.icon}`" :alt="related.name.chn">
              <span>{{ related.name.chn }}</span>
            </router-link>
          </div>
        </div>
      </div>
    </div>
  </div>
</template>
```

## 布局组件（MainLayout）

```vue
<template>
  <div class="main-layout">
    <ClassicNavigation />
    <div class="layout-body">
      <SidebarNav v-if="showSidebar" />
      <main class="layout-content">
        <slot />
      </main>
    </div>
  </div>
</template>

<script setup lang="ts">
import { computed } from 'vue';
import { useRoute } from 'vue-router';
import ClassicNavigation from '@/components/ClassicNavigation.vue';
import SidebarNav from '@/components/SidebarNav.vue';

const route = useRoute();

// 只在物品页面显示侧边栏
const showSidebar = computed(() => {
  return route.name === 'ItemCatalog' || route.name === 'ItemDetail';
});
</script>

<style scoped lang="scss">
.main-layout {
  min-height: 100vh;
  background: #000;
}

.layout-body {
  display: flex;
  padding-top: 150px; // 为固定导航留空间
  
  @media (max-width: 1000px) {
    padding-top: 6em;
  }
}

.layout-content {
  flex: 1;
  min-width: 0; // 防止内容溢出
}
</style>
```
